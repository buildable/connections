# Deletes a report

Deletes a report

**Requirements**

- Requires `reports` access scope.

[Documentation](https://shopify.dev/docs/api/admin-rest/2023-01/resources/report)

**Sample Payload**

```json
{
  "primaryResourceId": 132465798
}
```

**Note:** The `primaryResourceId` is the `id` of the report to delete.

**Sample Response**

```json
{}
```
