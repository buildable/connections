### Create Currency

Create currencies in Xero.

[Documentation](https://xeroapi.github.io/xero-node/accounting/index.html#api-Accounting-createCurrency)


**Sample Payload**

```json
{
  "currency": {
    "code": "USD",
    "description": "United States Dollar",
    "decimalPlaces": 2,
    "active": true,
    "default": true
  }
}
```

**Sample Response**
```json
{
  "currency": {
    "code": "USD",
    "description": "United States Dollar",
    "decimalPlaces": 2,
    "active": true,
    "default": true
  }
}
```
