### Create Contact Group

Create a new contact group in Xero.

[Documentation](https://xeroapi.github.io/xero-node/accounting/index.html#api-Accounting-createContactGroup)


**Sample Request**

```json
{
  "contactGroups": {
    "contactGroups": [
      {
        "name": "VIPs"
      }
    ]
  }
}
```

**Sample Response**

```json
{
  "contactGroups": [
    {
      "name": "VIPs"
    }
  ]
}
```
